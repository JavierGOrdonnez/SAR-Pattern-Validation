1. I made a class you can use inside the `class` folder. It also has images and logos. Just copy the class to another project if you want to use it.
2. mainbu.tex is the original .tex file for comparison. main.tex is the file you will have to compile. I haven't touched the contents, just the preamble (apart from TOC and bibliography, see later)
3. In the preamble you can put all your custom macros and the packages you need for a single report.
4. Table of contents: use \tableofcontents as usual, it will automatically start a new page
5. Bibliography: use \PrintBibliography{bib}, where "bib" is the .bib file name. It will automatically start a new page.
6. Class option:
* cover/nocover: it will print (or not) the cover page
* titlepage/notitlepage: it will print (or not) the title page
7. Metadata:
Example:

\DocuMetadata{
    Title={RF-Induced Heating of SENZA SCS Percutaneous Leads with a Single Wire Break during MR Examination},
    Subtitle={Projects CR-0838 and extensions},
    HeadTitle={RF Heating of Percutaneous Leads with a Single Wire Break},% if this is empty, the Title is used in the header
    Conducted={Lena Kranold, Tolga Goren, George Tsanidis, Theodoros Samaras},
    Date={Zurich, September 16, 2021},
    %Disclaimer={},% In case you want to change the default
    %Confidential={},% In case you want to change the default
    Project={0686},
    DocuName={CRR-686},
    Rev={0},
    AutName={Tolga Goren and Earl Zastrow},
    AutFunction={Project Leader, IT'IS},
    AutDate={2019.05.09},
    AppName={Niels Kuster},
    AppFunction={Quality Manager, IT’IS},
    AppDate={2019.05.09},
    Customer={Nevro Corp.},
    Watermark={DRAFT}
}

I believe they are self-explanatory. `HeadTitle` will print the title in the header: if empty, it will print `Title`.
`Conducted` also defines \authorname
`Date` also defines \reportdate
`Disclaimer` and `Confidential` have both default values you can override with the metadata.
`Watermark`: if left empty, the watermark is not printed.
